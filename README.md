# testing-bootcamp

This repository contains a .NET Blazor server application for testing bootcamp exercises.

## Requirements

- .NET 9 SDK or later

## Projects

### BootcampWebApp

A Blazor server application located at `src/BootcampWebApp/`.

To run the application:

```bash
cd src/BootcampWebApp
dotnet run
```

The application will be available at `http://localhost:5000` (or the port shown in the console output).

## Development

This repository is set up to support multiple projects in the `src/` directory for future expansion.
